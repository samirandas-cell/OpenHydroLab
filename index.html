OpenChannelLab

Physics-accurate, interactive animations for teaching open-channel hydraulics

OpenChannelLab is a free library of single-file, interactive HTML animations for undergraduate open-channel hydraulics. Every number on screen is computed live from the real governing equations — no faked motion. Each animation runs in any modern browser, needs no installation, and can be projected in class or explored by students at home.

Physics integrity

Each animation is built to a strict rule: every displayed quantity is computed from the governing equation (continuity, energy, momentum, Manning, the GVF equation), and results are verified by hand against worked textbook examples before release. These are teaching instruments, not illustrations.
